https://github.com/zaki3gh/note

リモート→ローカル
githubからclone
> git clone -v https://github.com/zaki3gh/note.git

 * アドレスは、Clone or downloadのコンボボックスを開くと分かる。

リモートリポジトリの変更を取り込む
> git pull origin ブランチ名
上流ブランチ(git status)で見えるものを取り込む場合は省略できる。
> git pull

ブランチの切り替え
>git checkout ブランチ名

リモートのブランチを上流ブランチとするローカルブランチの作成
> git branch ローカルのブランチ名  リモートのブランチ名
> git checkout ローカルのブランチ名
  (> git branch github origin/github
   > git checkout github)

ローカル
ファイルの追加
> git add ファイルパス

追加されたファイルの確認
> git status

追加されたファイルのcommit
> git commit -m "メッセージ"


ローカル→リモート
> git push origin ブランチ名
上流ブランチ(git status)で見えるところに入れる場合は省略できる。
> git push
